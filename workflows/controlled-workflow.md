# Controlled Workflow

## 每次任務標準流程

```text
1. Confirm Project Root
2. Read Core Docs
3. Select Task
4. Create Mini Plan
5. Implement Small Change
6. Run Focused Tests
7. Run Broader Tests
8. Review
9. Update todos/final/lessons
```

## Gate

### Gate A — 開始前

- [ ] 目前目錄正確。
- [ ] 已讀 AGENTS.md。
- [ ] 已讀 spec.md。
- [ ] 任務不跨專案。

### Gate B — 實作前

- [ ] 已列出會修改的檔案。
- [ ] 已確認是否涉及磁碟寫入。
- [ ] 若涉及 writer，必須有 Safety Agent 規格。

### Gate C — 完成前

- [ ] 測試通過。
- [ ] 文件更新。
- [ ] final.md 更新。
- [ ] 沒有真實磁碟寫入。
