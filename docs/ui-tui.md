# ui-tui.md — 中文 TUI 規格

## 設計目標

- 像 SPFDisk 一樣直覺。
- 低解析度終端可用。
- 鍵盤操作。
- 所有寫入前清楚提示。

## 主畫面

```text
┌──────────────────────────────────────────────┐
│ Rust SPFDisk                                 │
├──────────────────────────────────────────────┤
│ [1] 磁碟列表                                  │
│ [2] 檢視分割表                                │
│ [3] 快速分區精靈                              │
│ [4] 備份分割表                                │
│ [5] 還原分割表                                │
│ [6] 設定                                      │
│ [Q] 離開                                      │
└──────────────────────────────────────────────┘
```

## 預覽畫面

```text
即將建立以下分區：

#  名稱          大小       類型        格式
1  EFI System    512MiB     ESP         FAT32
2  MSR           16MiB      MSR         none
3  Windows       180GiB     Basic Data  NTFS/no-format
4  Recovery      1GiB       Recovery    NTFS/no-format
5  Data          330GiB     Basic Data  NTFS/no-format

[E] 編輯容量  [B] 備份  [W] 寫入  [Q] 取消
```

## 錯誤提示

錯誤訊息必須包含：

- 問題。
- 可能原因。
- 建議下一步。

範例：

```text
錯誤：磁碟空間不足，無法套用 Windows + D 槽模板。
原因：模板至少需要 80GiB，但目前可用空間只有 54GiB。
建議：縮小 Windows 分區，或改用手動分區。
```
